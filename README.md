# Mini Arcade (Mobile)

スマホ操作前提の、軽量ミニゲーム集です（HTML/CSS/JSのみ）。

## Games
- **Tetris**
- **Invaders**（Space Invaders風）

## 操作
### Tetris
- ◀ ▶ : 左右移動
- ⟳ : 回転
- ▼ : ソフトドロップ
- DROP : ハードドロップ

PCでも矢印キー / Space / Z / X / P が使えます。

### Invaders
- ◀ ▶ : 移動
- FIRE : 発射（PCは Space / ↑ / Z / X でもOK）
